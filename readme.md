# Scider

![Scider](./imgs/logo.png)

Scider是一款旨在帮助科研新手——尤其是本科生和低年级研究生——高效阅读、深度理解并系统性管理学术论文的智能辅助平台

## 项目结构预览
- `docs/`: 项目详尽文档、设计方案与技术调研
- `scripts/`: 开发、部署及数据处理相关的自动化脚本
- `src/`: 核心源代码
  - `backend/`: 后端服务
  - `frontend/`: 前端用户界面

## 快速开始
1. 克隆仓库: `git clone <repository-url>`
2. 安装依赖: 请参考 `src/backend` 和 `src/frontend` 下的具体说明
3. 运行项目: 查看各模块下的开发启动命令。
