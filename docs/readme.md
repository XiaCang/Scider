# 项目文档(Example)

本目录存储项目全生命周期的所有非代码类文档。

## 文档索引
1. **[Design]**: `architecture_design.md` - 系统架构图与模块划分。
2. **[API]**: `api_specification.md` - RESTful 接口定义与数据格式。
3. **[NABCD]**: `product_analysis.md` - 项目需求分析与市场价值评估。
4. **[Meeting]**: `meeting_minutes/` - 团队会议记录与任务分配。

## 贡献准则
所有设计变更需先在 docs 中更新，经由 Pull Request 评审通过后方可实施代码变更。
